# Phishing Analysis

This section documents practical phishing investigation exercises completed during my Blue Team Level 1 training.

The projects cover:

- Email structure and header analysis
- Phishing tactics and techniques
- Manual artifact extraction
- URL and domain investigation
- Attachment analysis
- File hashing
- Artifact reputation analysis
- Defensive response actions
- Phishing incident reporting

## Projects

### Manual Artifact Extraction

I manually extracted email, web, and file artifacts from two safe phishing email samples using Mozilla Thunderbird, Sublime Text, DomainTools, and Windows PowerShell.

[View project](manual-artifact-extraction/README.md)

### Investigating an Attachment

I investigated an HTML credential-harvesting attachment, decoded its source code, identified the Microsoft impersonation, and traced the credential-submission request with Chrome Developer Tools.

[View project](investigating-an-attachment/README.md)
